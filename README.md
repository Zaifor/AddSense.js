# AddSense.js
Async AdSense Banners via Javascript

Example:

```
AddSense.unit({
  container : ".view-banner",
  client : "ca-pub-xxxxxxxxxxxxxxxx",
  slot : "xxxxxxxxxx",
  width : 300,
  height : 250,
  overrideFormat : "true",
  pageUrl : "http://www.example.com",
  language: "en"
}).render();
```
